# Hole App
A perfectly normal application, doing nothing out of ordinary. Made using Svelte and Tauri.

## Prerequisites
- [pnpm](https://pnpm.js.org/en/installation)
- [rust](https://www.rust-lang.org/tools/install)

## Running
```sh
pnpm install
pnpm tauri dev
```
